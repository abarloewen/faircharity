# Faircharity e.V. — Website (Neuaufbau)

Komplett neu gebaute, zweisprachige (DE/EN) statische Website. Kein WordPress — nur HTML, CSS und etwas JavaScript. Schnell, sicher und einfach zu hosten.

## Öffnen
Doppelklick auf **`index.html`** öffnet die Seite im Browser. Alles funktioniert lokal ohne Server.

## Inhalt
14 Seiten: Home, Kurban, Patenschaft, Wasserversorgung, Hilfe zur Selbsthilfe, Zakat al-Māl, Fidya & Kaffara, Zakat al-Fitr, Eid Geschenk, Transparenz, Spenden, Kontakt, Impressum, Datenschutz.

## Funktionen
- **Sprachumschalter DE/EN** oben rechts (merkt sich die Wahl).
- **Zakat-Rechner** (zakat.html) und **Fidya-/Kaffara-Rechner** (fidyakaffara.html).
- Responsives Menü, mobiles Drawer-Menü, sanfte Animationen.
- Neues Logo + Wappen (`assets/img/logo*.svg`) und maßgeschneiderte Illustrationen statt Stockfotos.

## Vor dem Livegang anpassen
1. **Impressum** und **Datenschutz**: Platzhalter `[ … ]` mit echten Vereinsdaten füllen (Adresse, Vorstand, Vereinsregister, Finanzamt).
2. **Kontaktformular** (kontakt.html) mit E-Mail-/Formular-Dienst verbinden — sendet aktuell keine Daten.
3. **Google Fonts** für maximalen Datenschutz lokal hosten (siehe Datenschutz-Seite).
4. Echte Projektfotos in den Card-/Split-Bereichen ergänzen, wenn gewünscht.

## Hosting
Ordner als Ganzes zu einem Static-Host hochladen (z. B. Netlify, Vercel, oder klassisches Webhosting). Keine Datenbank nötig.
